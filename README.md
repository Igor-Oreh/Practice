# Practice

## Последовательность команд Git


### Создание репозитория

```bash
git init
git remote add origin https://github.com/Igor-Oreh/Practice.git
```
### Создание веток
```bash
git checkout -b homework-1-sql
git checkout -b homework-2-sql
git checkout -b homework-3-swagger
git checkout -b homework-4-postman
git checkout -b homework-1-Net
git checkout -b homework-2-Net
git checkout -b homework-3-ASP.NET
git checkout -b homework-4-EF-core
```
